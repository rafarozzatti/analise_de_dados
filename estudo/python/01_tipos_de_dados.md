# Tipo Inteiro (`int`)

Números inteiros, sem parte decimal.

```python
print(1 + 2)
```

**Saída:**

```
3
```

---

# Tipo Flutuante (`float`)

Números com parte decimal (reais).

```python
print(2.5 + 7.6)
```

**Saída:**

```
10.1
```

---

# Tipo Booleano (`bool`)

Valores lógicos, que representam **verdadeiro** (`True`) ou **falso** (`False`).

```python
print(True)
print(False)
```

**Saída:**

```
True
False
```

---

# Tipo String (`str`)

Textos entre aspas simples ou duplas.

```python
print('Python')
```

**Saída:**

```
Python
```
